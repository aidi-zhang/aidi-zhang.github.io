project 1

a project with a background image

每个项目都有一个独立的展示页面，支持灵活的三栏图片布局，图片宽度可设为 1/3、2/3 或全宽。

如需为项目卡片设置封面图，只需在 front matter 中添加 img 字段，如下所示：

---
layout: page
title: project
description: a project with a background image
img: /assets/img/12.jpg
---
图片说明示例：左图为穿越隧道的公路，中图为艺术感十足的落叶特写，右图为抓握松针的伐木工人。
这张图片同样可以添加说明文字。

图片行之间也可以插入正文段落，用于介绍项目背景、描述研究过程，再以下一组图片收尾，展示最终成果。

也可以使用 2/3 + 1/3 的非对称图片布局，如上图所示。

代码非常简洁：将图片包裹在 <div class="col-sm"> 中，再放入 <div class="row"> 即可（参见 Bootstrap 栅格系统）。添加 img-fluid 使图片自适应，rounded 和 z-depth-1 分别为图片添加圆角和阴影。上方最后一行图片的代码如下：

<div class="row justify-content-sm-center">
    <div class="col-sm-8 mt-3 mt-md-0">
        {% include figure.html path="assets/img/6.jpg" title="example image" class="img-fluid rounded z-depth-1" %}
    </div>
    <div class="col-sm-4 mt-3 mt-md-0">
        {% include figure.html path="assets/img/11.jpg" title="example image" class="img-fluid rounded z-depth-1" %}
    </div>
</div>
